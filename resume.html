Step 2: Create the Apps Script "Backend"

In the MyPortfolio folder, create a new **Google Apps Script** (New > More > Google Apps Script). Replace the code with the script provided in the response.

You MUST replace the **`PORTFOLIO_FOLDER_ID`** in the script with the ID of your main **MyPortfolio** folder.

Step 3: Deploy the Script as a Web App (API)

In the Apps Script Editor, click **Deploy** > **New Deployment**:

  • **Type:** Web App
  • **Execute as:** Me
  • **Who has access:** Anyone

Click **Deploy** and **COPY THE WEB APP URL**.

Step 4: Update the HTML Code (Crucial)

Paste the **Web App URL** you copied into the JavaScript section of this HTML file, replacing the placeholder:

let portfolioConfig = { webAppUrl: 'YOUR_COPIED_APPS_SCRIPT_URL_HERE' };

Once updated, refresh this page.
